# Sistema de Gestión de Congregación (SGC)

Este sistema modular permite la gestión integral de una congregación, dividiéndose en áreas principales que comparten una base de datos común de personas y un backend centralizado.

## Índice de Documentación

1.  [**Módulo de Administración**](./SGC_Administracion.md): Gestión centralizada de la base de datos de personas y control de acceso de usuarios.
2.  [**Módulo de Reuniones**](./SGC_Reuniones.md): Gestión de programas, plantillas y discursos públicos.
3.  [**Definición del Backend**](./SGC_Backend.md): Arquitectura de datos, seguridad, 2FA y cifrado de conocimiento cero.
4.  [**Base de Datos de Personas**](./SGC_Personas.md): Especificación técnica de la ficha de datos compartida y campos cifrados.
5.  **Módulos Futuros**:
    -   [Predicación](./SGC_Predicacion.md)
    -   [Salón del Reino](./SGC_Salon.md)

## Tecnologías e Interfaz
-   **Despliegue:** GitHub Pages (Front-end).
-   **Estilo:** Tailwind CSS + Material Design.
-   **Iconografía:** Material Icons.
-   **Adaptabilidad:** Diseño Responsive y Mobile First.
-   **Tematización:** Modo Oscuro y Claro nativo.
