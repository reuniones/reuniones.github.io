download 1. Descargar reloj.bin open_in_new 2. Ir a la página del reloj
Pasos en la página del reloj:
  1. Buscá la sección llamada "OTA Update".
  2. Hacé clic en el botón para elegir archivo.
  3. Seleccioná el archivo reloj.bin descargado.
  4. Hacé clic en "Update" y esperá al 100%.

Buscando conexión con el reloj

No se puede establecer una conexión con el reloj. Esto puede deberse a problemas de red, configuración o restricciones de seguridad del navegador. La siguiente información puede ayudarte a diagnosticar el problema:

Direcciones intentadas:

Las siguientes acciones pueden ayudarte a resolver el problema:

Asegurate de que el reloj esté encendido y conectado a la misma red que este dispositivo.

Podés verificar que el reloj se conectó si muestra la hora actualizada.

Si cambió la configuración del wifi, acabás de instalar el reloj o actualizaste el firmware, seguí estos pasos:
  1. Buscá la red WiFi "Reloj" y conectate.
  2. Entrá a http://192.168.4.1 para configurar el WiFi.

Por seguridad el navegador bloquea conexiones a dispositivos dentro de la red local o sin encriptación. Seguí las siguientes instrucciones para permitir la conexión en tu navegador.

Extensiones como uBlock o AdBlock pueden interferir con las conexiones locales. Desactivalos para este sitio.

Si verificaste los pasos anteriores pero sigue sin conectarse, podés intentar ingresar la dirección IP del reloj en el diálogo de configuración. Ahí también se puede escanear la red para encontrar la dirección IP del reloj.

Tené en cuenta que la dirección IP puede cambiar con el tiempo.

Sólo después de intentar todos los pasos anteriores varias veces, contactá al desarrollador para obtener soporte técnico.

El sistema seguirá intentando conectar automáticamente.
